# Discovery — Modern CLI Tools

## gh
Liste et gère les repos, issues et PRs GitHub depuis le terminal.
Surprenant : on peut créer un repo et le cloner en une seule commande.
Use case : remplacer complètement l'interface web GitHub.

## gh copilot
Traduit du langage naturel en commandes shell via GPT.
Surprenant : il peut exécuter la commande directement sur ta machine.
Use case : générer des commandes complexes sans les mémoriser.

## bat
Remplacement de `cat` avec coloration syntaxique.
Surprenant : affiche les numéros de ligne et le nom du fichier automatiquement.
Use case : lire du code dans le terminal confortablement.

## delta
Meilleur affichage des diffs git.
Surprenant : affichage côte à côte avec couleurs.
Use case : relire ses changements avant un commit.

## fzf
Fuzzy finder interactif pour n'importe quelle liste.
Surprenant : fonctionne en pipe avec n'importe quelle commande.
Use case : chercher dans l'historique git interactivement.
